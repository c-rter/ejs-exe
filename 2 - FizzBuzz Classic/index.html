FizzBuzz Classic - Division

Enter a value for Fizz, and Buzz. The function will list numbers from 1 - 100. If the number is divisible by the Fizz value, it will be replaced with Fizz. If the number is divisible by the Buzz value, it will be replaced with Buzz. Numbers that are divisible by both will be replaced with FizzBuzz.
